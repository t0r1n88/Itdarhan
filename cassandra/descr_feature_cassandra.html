Обработка выгрузки по вакансиям с портала Работа в России

С портала Работа в России по ссылке https://trudvsem.ru/opendata/datasets скачивается файл c вакансиями в формате csv для вашего федерального округа.

После обработки с помощью программы создается файл в формате xlsx(Excel) в котором в удобном для работы виде отображены вакансии для вашего региона

В файле находятся 7 листов с вакансиями отобранными по определенным условиям

  • Только подтвержденные вакансии - на этом листе находятся вакансии, у которых в колонке Статус проверки вакансии указано значение Одобрено;
  • Квотируемые - на этом листе находятся квотируемые вакансии региона (вакансии, у которых в колонке Квотируемое место указано значение Квотируемое место);
  • Для соц категорий - на этом листе находятся вакансии, у которых в колонке социально защищенная категория стоит какое-то значение (инвалиды, беженцы, родители инвалидов и т.п., граждане, уволенные с военной службы);
  • С предоставлением жилья - на этом листе находятся вакансии, у которых в колонке Жилье от организации стоит значение Предоставляется жилье;
  • Трудовая мобильность - – на этом листе указаны вакансии, у которых в колонке Программа трудовой мобильности указано значение Вакансия по программе трудовой мобильности;
  • Для иностранцев - на этом листе указаны вакансии, у которых в колонке Для иностранцев указано значение Возможно трудоустройство иностранных граждан;
  • Вакансии со всеми статусами - на этом листе указаны вакансии со всеми статусами включая отклоненные и т.п.

На основании списка размещенного на листе Только подтвержденные вакансии программа подсчитывает количество вакансий в разрезе:

  • Отраслей
  • Муниципалитетов
  • Муниципалитетов и отраслей
  • Отраслей и муниципалитетов
  • Работодателей
  • Образованию и отраслям
  • Образованию и работодателям
  • Графику работы и отраслям
  • Графику работы и работодателям
  • Типу занятости и отраслям
  • Типу занятости и работодателям
  • Квотируемые места по отраслям
  • Квотируемые места по работодателям
  • По соцкатегориям
  • Требуемому опыту и отраслям
  • Требуемому опыту и работодателям
  • По отраслям и категориям минимальной предлагаемой заработной платы
  • По работодателям и категориям минимальной предлагаемой заработной платы

Подсчитываются средние и медианные значения минимальной предлагаемой заработной платы

  • По отраслям
  • По работодателям и отраслям

Аналогичная статистика создается на основании каждого из оставшихся листов (Квотируемые, для соц категорий, для иностранцев и т.д.)

Дополнительные опции при обработке выгрузки

Можно создать список работодателей для которых параллельно с основными будут создаваться файлы с вакансиями и статистикой только по выбранным работодателям.

Можно создать список условий по которым будут отфильтрованы данные перед обработкой.

Например, если использовать файл показанный на скриншоте выше, то сначала будут отобраны все вакансии у которых в колонке Сфера деятельности указаны отрасли перечисленные в первой колонке, затем из отобранных вакансий будут отобраны те, у которых в колонке Муниципалитет встречаются перечисленные во второй колонке значения, после этого останутся только те вакансии у которых в колонке Вакансия встречаются указанные в третьей колонке значения. По оставшимся после такой фильтрации вакансиям и будет производиться подсчет статистики и создание списка вакансий.

Можно создать список со словами которые должны содержаться в названии вакансии, например врач, учитель, сварщик. В этот список во вторую колонку можно добавить слова которых не должно быть в названии вакансии. Программа параллельно с основными файлами будет создавать файлы где будут содержаться только отфильтрованные вакансии.

Сравнение изменений рынка вакансий по двум выгрузкам на основе аналитики, полученной с помощью Кассандры

Используя два файла со сводами по вакансиям за разные даты, можно получить разницу в количестве вакансий по отраслям, работодателям, муниципалитетам и т.п. Самое главное этот инструмент позволяет отслеживать изменения количества рабочих мест по конкретным вакансиям. Для каждого значения подсчитывается разница, абсолютная разница, изменение в процентах, процентное соотношение. Строки, где количество рабочих мест уменьшилось выделены оранжевым, где увеличилось зеленым.

Создания временных рядов для отображения динамики изменения количества вакансий по основным показателям

На основании файлов созданных Кассандрой можно создать временные ряды для отслеживания динамики ситуации с количеством вакансий по отраслям, муниципалитетам, работодателям, требуемому образованию, графику работы, требуемому опыту и т.п.
Временные ряды создаются на основе 2 видов файлов создаваемых Кассандрой:

  • Файлов сводов содержащих подсчеты количества вакансий в разрезе(отраслей, муниципалитетов, работодателей и т.п.)
  • Файлов содержащих списки вакансий

Алгоритм обработки прост. В одну папку собираются файлы за интересующие вас даты. Файлы в названиях должны содержать дату в формате DD_MM_YYYY например 15_06_2026.

При обработке файлов сводов временные ряды создаются в трех видах:

Вертикальный вид, когда значения временного ряда расположены сверху вниз в хронологическом порядке

Горизонтальный вид, когда значения временного ряда расположены слева направо в хронологическом порядке

Для сводов, предназначенный для использования в сводных таблицах.

При обработке файлов содержащих списки вакансий происходит объединение этих списков в один большой список на основе которого удобно создавать сводные таблицы и дашборды. При использовании файла с отслеживаемыми вакансиями можно добавляется дополнительная колонка с названием отслеживаемой вакансии

Очистка списка от ошибок

С ее помощью вы можете провести первичную обработку списка xlsx. Удалить лишние пробельные символы в начале и конце каждой ячейки в таблице, а также между словами.

Для колонок содержащих в названии слова Фамилия,Имя,Отчество, ФИО, сделать так чтобы каждое слово в этих колонках начиналось с большой буквы.

Для колонок название которых содержит слово дата найти неверно заполненные ячейки.

Для колонок название которых содержит слово снилс привести к данные к стандартному виду ХХХ-ХХХ-ХХХ ХХ и найти неверно заполненные ячейки (где больше или меньше цифр чем надо).

Для колонок название которых содержит слово инн привести к формату из 12 цифр (ИНН физического лица) удалив все символы кроме цифр и найти неверно заполненные ячейки (где больше или меньше цифр чем надо).

Для колонок название которых содержит словосочетания (в любом порядке): серия паспорт, номер паспорт, код подразд привести их к стандартному виду и найти неверно заполненные ячейки (где больше или меньше цифр чем надо)..

Для колонок название которых содержит слово телефон удалить все нечисловые символы.

Для колонок названия которых содержат в себе слова электрон почта, e-mail удалить все пробельные символы.

Найти дубликаты в каждой из колонок таблицы с указанием на какой строке оригинальной таблицы они находятся.

Найти дубликаты по нескольким колонкам таблицы с указанием на какой строке оригинальной таблицы они находятся.

Подсчитать сколько раз то или иное значение встречается в каждой из колонок.

Найти случаи смешения русских и английских букв в одном слове с указанием на какой строке они находятся и перечислением какие именно буквы являются смешанными.

Создание однотипных документов docx (Word и его аналоги)

Для примера у вас есть список группы из 100 обучающихся на курсе и вам нужно на каждого из них сделать справку, договор, согласие и т.п. при этом разделив создаваемые документы по папкам (чаще всего такое пригодится при создании pdf версий удостоверений и свидетельств для рассылки). Документы отличается только паспортными данными и реквизитами, и чтобы вам не пришлось 100 раз создавать документ на каждого, вы можете использовать эту программу. Вам нужно будет только указать таблицу с используемыми данными формата Excel и создать шаблон документа в Word указав в этом шаблоне в двойных фигурных скобках {{}} названия колонок из файла с расширением xlsx. После чего программа, беря данные из таблицы и подставляя эти данные в {{}} в шаблоне, создаст вам столько документов сколько строк есть в таблице с сохранением форматирования шаблона. 

Можно создавать pdf версии создаваемых документов, можно создавать объединенные файлы (т.е. вместо 100 отдельных документов у вас будет один файл в котором будет находиться 100 документов).

Также можно задавать структуру папок в которых будут сохраняться документы. Достаточно просто указать не более трех порядковых номеров колонок по которым вы хотите рассортировать документы
