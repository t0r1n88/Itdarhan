Список изменений в программе Кассандра

 

  • Версия 7.1. Добавлена вкладка с дополнительной информацией (номер свидетельства о государственной регистрации программы для ЭВМ, сведения о лицензии и используемых библиотеках, список изменений, ссылки для скачивания). В названиях создаваемых файлов теперь указывается предыдущий день, это сделано чтобы избежать путаницы поскольку на сайте trudvsem.ru выкладываются датасеты с вакансиями за предыдущий день.
  • Версия 7.0. Добавлена функция создания временных рядов из файлов сводов и списков вакансий по данным с портала Работа в России. С помощью этой функции можно проследить динамику изменения основных показателей (общее количество вакансий в регионе, количество вакансий по отраслям, муниципалитетам, работодателям, определенным вакансиям и т.п.).Функции обработки мониторингов трудоустройства выпускников перенесены на одну вкладку.
  • Версия 6.6. Добавлена обработка мониторинга занятости выпускников за сентябрь 2025. Добавлены проверки наличия и дублирования уникальных кодов для выпускников, указанных на листах Нозологии и Целевики. Добавлены проверки для статусов выпускников целевиков.
  • Версия 6.5. Добавлена обработка мониторинга занятости выпускников за май 2025.Добавлено создание файла с профессиями/специальностями, имеющими одинаковый код, но разные названия для облегчения обработки результатов.В функции Свод Работа в России сделана необязательной кнопка выбора файла с данными работодателей, добавлено две новых сводных таблицы – Количество вакансий по муниципалитетам в разрезе сфер деятельности и Количество вакансий по сферам деятельности в разрезе муниципалитетов. В списках вакансий добавлено отображение email контактного лица (раньше отображался только email работодателя).
  • Версия 6.4. Добавлена обработка двух редких случаев. 1) Когда у последней специальности в таблице были удалены строки (в прежней версии такие данные не попадали в обработку); 2) Обработка случая, когда во всех файлах найдены ошибки (в прежней версии программа вылетала с ошибкой).
  • Версия 6.3. В связи с добавлением новых колонок в форму по нозологиям (15 строк) обновлена функциональность программы.
  • Версия 6.2. Теперь можно задавать параметры предварительной обработки файла с помощью создания специального файла в формате xlsx. Используя указанные в таком файле значения, программа отфильтровывает значения, можно использовать до трех колонок и любое количество значений для выбранных колонок. Отфильтрованные данные используются для дальнейшего создания аналитики и списков.
  • Версия 6.1. Кассандра теперь корректно обрабатывает файлы вакансий csv с новой структурой. В файл Вакансии по региону добавлены 3 новых листа: Вакансии с предоставлением жилья, Вакансии участвующие в программе мобильной занятости, Вакансии где могут работать иностранные специалисты.Теперь создаются дополнительные сводные файлы по новым листам.
  • Версия 6.0. Добавлена возможность сбора и обработки мониторинга занятости выпускников 2024 для сервиса «Система сбора и синхронизации ресурсов» (https://data.firpo.ru).
  • Версия 5.5. Добавлена колонка Муниципалитет. В этой колонке находится муниципалитет к которому принадлежит вакансия.
  • Версия 5.4. Изменен формат программы, теперь это архив, который нужно разархивировать. Это было сделано для того, чтобы уменьшить количество ложных срабатываний антивирусов. В файле с вакансиями добавлены колонки: Категория минимальной зарплаты (для удобства аналитики), Краткое название работодателя (длинные названия превращаются в аббревиатуры)
  • Версия 5.3. Улучшен механизм отслеживания динамики изменения вакансий.
  • Версия 5.2. В создаваемые списки с вакансиями добавлены колонки: Ссылка на вакансию (прямая ссылка на вакансию), ID вакансии, ID работодателя.
  • Версия 5.1. Добавлено создание списка квотируемых вакансий и списка вакансий для социальных категорий (инвалиды, беженцы и т.п.). Исправлена ошибка при подсчете средних значений зарплаты.
  • Версия 5.0. Добавлена возможность обработки данных с сайта Работа в России. Добавлена возможность отслеживать динамику изменений в данных по вакансиям.
  • Версия 4.1. Добавлена возможность обработки формы 3 мониторинга занятости выпускников ожидаемый выпуск.
  • Версия 4.0. Обновлен процесс обработки данных в связи с изменением исходных форм мониторинга.
  • Версия 3.6. Добавлена версия для Linux систем (отечественные ОС, Ubuntu)
  • Версия 3.5. Добавлена обработка измененной формы сбора данных по нозологиям(15 строк). Теперь создается файл в котором описаны названия листов откуда берутся данные из файлов мониторинга (это сделано поскольку некоторые СПО добавляют в файл с данными за 2022 лист с данными за 2023)
  • Версия 3.4. Добавлена обработка отчетов ОПК по отраслям (2 строки на специальность, 2 заполняемых листа)
  • Версия 3.3. Добавлена обработка случаев неправильного заполнения колонки 03 Код специальности. Программа теперь проверяет чтобы в этой колонке был только код специальности в формате XX.XX.XX . Если там находятся даты, название специальности вместе с кодом и т.п. программа не обрабатывает такой файл и записывает его название в файл Ошибки для последующего исправления. Программа проверяет чтобы код специальности был заполнен на всех 15 строках для каждой специальности если это условие не соблюдено, то такой файл тоже не обрабатывается.
  • Версия 3.2. Мелкие доработки, связанные с описанием ошибок.
  • Версия 3.1. Добавлена возможность сравнивать 2 одинаковых по размеру таблицы. Это может помочь при проверке регулярности заполнения таблиц СПО.
