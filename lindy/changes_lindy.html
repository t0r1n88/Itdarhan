Список изменений в программе Линди

 

Версия 2.5 – Программа добавлена в реестр отечественного программного обеспечения.
Добавлена функция для создания прочих документов (полностью повторяет функционал аналогичной функции из Весты).
Добавлена автоматическая конвертация пола при создании файла ФИС-ФРДО.
Исправлена ошибка возникавшая в случае незаполнения колонки Дата_рождения.

Версия 2.4 Добавлено автоматическое создание документов для несовершеннолетних и совершеннолетних слушателей при наличии соответствующих шаблонов.Добавлено автоматическое склонение частей ФИО. Добавлена обработка случаев когда у слушателя отсутствует отчество.

Версия 2.3 Добавлена вкладка О программе. Добавлены ссылки.
