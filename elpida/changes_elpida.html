Список изменений в программе Эльпида

 

Версия 1.4 - Добавлена возможность при использовании функции Обработка списка искать дубликаты по нескольким выбранным колонкам. Добавлено создание файла с частотными таблицами (сколько раз то или иное значение встречается в ней) для каждой колонки при использовании функции Обработка списка.Исправлена ошибка в функции Создание документов при сохранении по папкам в случае если в значении по которому создается название папки был пробел в начале или в конце. Исправлена ошибка в функции Разница двух таблиц появлявшаяся в случае если различия между файлами отсутствовали. Для всех функций добавлена возможность работы с файлами с расширением xlsm.

Версия 1.3 - Добавлена функция Выверка ЕГИССО, предназначенная для поиска дублей в файлах выверки фактов назначений по пересекающимся периодам. В функцию Подготовка данных для ЕГИССО добавлены поиск дубликатов и создание частотных таблиц по каждой колонке шаблона ЕГИССО. Добавлена проверка соответствия категории categoryID значению LMSZID. В общем файле для облегчения проверки добавлены колонки с текстовым наименованием LMSZID и categoryID.

Версия 1.2 - Добавлена функция структурирования данных из таблиц (из учебных планов, рабочих программ и т.п.). Исправлена ошибка когда при создании pdf файлов в названии созданного файла не писался тип документа.

Версия 1.1 – Добавлена функция поиска и исправления ошибок в файлах ЕГИССО(даты,ФИО,СНИЛС, проверка правильности заполнения связных колонок и т.п.). Добавлена функция массового пакетного создания документов из комплекта шаблонов. Добавлена функция подсчета текущего возраста на определенную дату, вычисления возрастной категории (совершеннолетний/несовершеннолетний, по годам, месяцам).

Версия 1.0 - Базовая версия.
