Список изменений в программе Веста

 

Версия 1.56 – Добавлена новая функция Обработка отчетов с помощью которой можно суммировать данные из большого количества отчетов в один путем сложения значений ячеек.При этом создается файл в котором можно проверить как получилось то или иное число в итоговом файле с указанием названия исходного файла и строки откуда были взяты данные. При работе функции Слияние файлов название файла откуда были взяты данные теперь отображается в первой колонке, а не в последней.

Версия 1.55 – Исправлена ошибка при разделении таблицы когда в колонке есть 2 одинаковых значения отличающихся только регистром. Улучшена обработка ошибок возникающих при попытке обработать поврежденные файлы xlsx.

Версия 1.54 Добавлена опция создания только PDF документов и сохранения их по вложенным папкам (на основе категорий выбранных колонок таблицы) при работе функции Создание документов.

Версия 1.53 Добавлена опция сохранения документов по вложенным папкам (на основе категорий выбранных колонок таблицы) при работе функции Создание документов .Добавлено создание категорий Совершеннолетний/Несовершеннолетний при работе функции Обработка дат рождения. Исправлена ошибка с некорректным отображением исходной колонки с обрабатываемой датой при работе функции Обработка дат рождения. Добавлена возможность склонения ФИО состоящих из Фамилии и Имени при работе функции Склонение ФИО по падежам.

Версия 1.52. Изменения в графическом интерфейсе.

Версия 1.51. Теперь вместо exe файла программа представляет собой заархивированную папку. Это было сделано для того, чтобы уменьшить количество ложных срабатываний антивируса. Обработка списка – добавлен поиск слов, в которых смешаны русские и английские слова.

Версия 1.50. Добавлена версия для Linux. Исправлен интерфейс. Теперь при создании документов из шаблонов пробелы не заменяются нижним подчеркиванием, добавлено ограничение на количество символов в названиях создаваемых файлов из шаблонов (80 символов) все что превышает этот порог обрезается.

Версия 1.42. Теперь при создании сводной таблицы при помощи функции Сводные таблицы на листе Количество вычисляются процентные соотношения внутри группы.

Версия 1.41. В функцию Обработка списков добавлен чекбокс Проверить колонки на дубликаты. Это нужно для большей гибкости в обработке поскольку данный процесс занимает много времени и не всем пользователям нужен.

Версия 1.40. Добавлена новая функция Сводные таблицы. С ее помощью вы сможете быстро создать сводные таблицы по выбранным вами колонкам таблицы. Вы вводите порядковые номера колонок, по которым вам нужен свод, вводите порядковые номера колонок, по которым вам нужно произвести вычисления. Программа создает файл Excel в котором на отдельных листах находятся сводные таблицы по следующим вариантам группировки:
Сумма, Среднее, Медиана, Минимум, Максимум, Количество значений, Самое частое значение, Количество самых частых значений, Количество дубликатов.
При использовании функции Обработка Списка теперь создается еще один файл Excel. В этом файле для каждой колонки обрабатываемой таблицы создается лист с перечислением дубликатов содержащихся в этой колонке.
В колонках в названиях которых есть слова: ФИО, Фамилия ,Имя, Отчество проводится поиск символов латинского алфавита чтобы избежать ошибок с [oeacpHxM]

Версия 1.39. Добавлена новая функция Обработка списка. С ее помощью вы можете провести первичную обработку списка Excel. Удалить лишние пробельные символы в начале и конце каждой ячейки в таблице.
Для колонок содержащих в названии слова Фамилия,Имя,Отчество, ФИО убрать лишние пробелы между словами, сделать так чтобы каждое слово в этих колонках начиналось с большой буквы.
Для колонок название которых содержит слово дата найти неверно заполненные ячейки.
Для колонок название которых содержит слово снилс привести к данные к стандартному виду ХХХ-ХХХ-ХХХ ХХ и найти неверно заполненные ячейки.
Для колонок название которых содержит слово инн привести к формату из 12 цифр удалив все символы кроме цифр и найти неверно заполненные ячейки
Для колонок название которых содержит словосочетания (в любом порядке): серия паспорт, номер паспорт, код подразд привести их к стандартному виду и найти неверно заполненные ячейки.
Для колонок название которых содержит слово телефон удалить все нечисловые символы.
Для колонок названия которых содержат в себе слова электрон почта,e-mail удалить все пробельные символы.

Версия 1.38. Добавлена новая функция Разделение таблицы. С её помощью вы можете разбивать большие таблицы на более мелкие, исходя из значений, которые находятся в выбранной колонке таблицы. Это означает, что программа создаст столько новых листов (или файлов), сколько уникальных значений имеется в выбранной колонке. Это поможет упорядочить и разделить большие объемы данных на более удобные части для анализа или хранения.

Версия 1.37. При объединении 2 таблиц дубликаты, в колонках, которые используются для объединения копируются в отдельные файлы(дубликаты с учетом регистра и пробелов и дубликаты без учета регистров и пробелов).
Добавлена обработка неверного оформления шаблона для генерации однотипных документов. Теперь если в шаблоне не хватает фигурных скобок или в названии колонки вставляемой в документ есть пробелы или цифры будет демонстрироваться окно с ошибкой.

Версия 1.36. При объединении 2 таблиц, колонки по которым ведется объединение обрабатываются 2 способами: без учета пробелов и регистра (т.е. ИВАНОВ ИВАн и Иванов Иван будут считаться одинаковыми); с учетом пробелов и регистра (т.е. Иванов Иван и иванов Иван будут считаться разными). Итоговые файлы создаются с учетом этих различий.

Версия 1.35. Теперь при объединении 2 таблиц в колонках, по которым ведется объединение можно не обращать внимание на разницу в написании строчных и прописных букв. Т.е. Иванов и ИВАНОВ будут корректно обработаны.

Версия 1.34. Добавлена возможность создания pdf документов.

Версия 1.33. Исправлена ошибка, связанная с некорректной обработкой колонок содержащих данные с плавающей точкой при объединении 2 таблиц.

Версия 1.32. Теперь при сравнении двух таблиц Веста дополнительно считает значение разницы между значением в первой и во второй таблице, % соотношение и насколько в процентах изменились значения.

Версия 1.31. Добавлена новая функция сравнения 2 таблиц. Теперь Веста может найти разницу между двумя таблицами и показать значения которые отличаются.

Версия 1.30. Исправлена проблема возникавшая при генерации однотипных документов с названиями содержащими запрещенные символы (“:|/\?*) например названия организаций. При сохранении таких документов запрещенные символы заменяются на пробел. Исправлено сохранение однотипных документов с одинаковым названием, теперь в таком случае к названию прибавляется цифровой индекс.

Версия 1.29. Исправлена проблема с добавлением 0:00:00 в идентификатор объединения возникающая при объединении 2 таблиц, когда в качестве колонки объединения выбрана дата, например ФИО + Дата рождения. Для того чтобы этой проблемы не было, название колонки с датой используемое для создание идентификатора объединения должно содержать слово Дата или дата.

Версия 1.28. При слиянии большого количества таблиц если в таблицах больше 3 колонок, программа удалит пустые строки и строки, в которых заполнена только одна колонка. Это позволит избежать добавления в общую таблицу строк, у которых, например заполнены только порядковые номера.

Версия 1.27. При склонении ФИО теперь создаются дополнительные колонки с инициалами в формате Иванов И.И. или И.И. Иванов. Также создаются колонки с пробелом после имени в формате Иванов И. И. или И. И. Иванов. Создаются колонки с склонением по падежам фамилий с инициалами.
Исправлена ошибка, когда код специальности при создании однотипных документов превращался в дату.

Версия 1.26. Добавлена возможность создания документа из шаблона для конкретного ФИО(значения). В общей таблице образующейся при слиянии большого количества файлов помимо названия файлов откуда взяты значения теперь указывается порядковые номера строк.

Версия 1.25. Удалена принудительная обработка дубликатов при слиянии 2 таблиц с целью обеспечения правильной работы при добавлении новых колонок. Исправлена ошибка, когда при слиянии 2 таблиц по коду специальности происходила конвертация кодов специальностей в дату.

Версия 1.24. Теперь подсчет частоты значений и базовых статистик в функции Подсчет данных ведется сразу по всем колонкам таблицы. В итоговом файле создается столько листов сколько есть колонок в обрабатываемой таблице. На каждом листе записывается частота значений в соответствующей колонке или базовая статистика по этой колонке.

Версия 1.23. Теперь при обновлении данных одной таблицы данными из другой таблицы, пустые ячейки из другой таблицы не будут стирать данные в обновляемой таблице.

Версия 1.22. Добавлена возможность обновления данных таблицы данными из другой таблицы. При совпадении выбранных параметров (например ФИО) данные в колонках с одинаковыми названиями будут обновлены, а если совпадений нет, то данные из второй таблицы будут добавлены в нужные колонки, а также будут добавлены колонки из второй таблицы, которых нет в первой таблице. Теперь можно будет легко обновлять данные в больших таблицах.
Добавлена возможность вставки в текстовые поля из буфера обмена.

Версия 1.21. Исправлена ошибка, при которой при слиянии файлов с помощью варианта А и Б терялась первая строка после заголовка.

Версия 1.20. При слиянии 2 таблиц, в итоговый файл теперь добавляется лист с названием Объединённая таблица, в нем находятся данные обеих таблиц вне зависимости от их совпадения. Это сделано для случаев когда нужно последовательно добавить к большой таблице несколько маленьких. Исправлена ошибка изредка возникавшая, когда итоговую таблицу, полученную при слиянии 2 таблиц, использовали для слияния с новой таблицей.

Версия 1.19. Переработано слияние файлов. Теперь можно объединять не по одному листу из каждого файла, а сразу все. На выбор 3 варианта слияния: по названию листов, по порядку листов, по названию листов с указанием количества строк заголовка.

Версия 1.18. Добавлена возможность склонения ФИО по падежам (родительный, дательный, винительный, творительный, предложный). При создании однотипных документов из шаблона незаполненные ячейки из таблицы будут отображаться в итоговых документах в виде пробелов (раньше отображались как nan).

Версия 1.17. Убраны лишние колонки в итоговом файле, который создается при Слиянии 2 таблиц. Т.е. теперь на листах с уникальными данными каждой из таблиц не будет пустых колонок из другой таблицы.

Версия 1.16. Переработана обработка колонок с датами, если они используются при слиянии 2 таблиц.

Версия 1.15. Переработана функция сравнения и объединения 2 таблиц, теперь она называется Слияние 2 таблиц. Теперь можно объединять таблицы по нескольким колонкам.

Версия 1.14. Добавлена возможность слияния в один список большого количества маленьких таблиц с одинаковой структурой. Улучшена функциональность объединения 2 таблиц по выбранным колонкам. Теперь для корректного объединения из таблиц удаляются дубликаты. Дубликаты из каждого файла сохраняются на отдельные листы в итоговом файле.

Версия 1.13. При генерации файлов в их названиях убрано время создания. Текущая версия программы отображается в заголовке окна программы.

Версия 1.12. Добавлена возможность сохранения генерируемых из шаблона файлов в один файл, с целью облегчения печати и редактирования.

Версия 1.11. Добавлена новая вкладка Извлечение данных для подсчета значений ячеек в нескольких файлах Excel имеющих одинаковую структуру (отчеты, анкеты и т.п.)

Версия 1.10. Во вкладке Обработка дат теперь можно выбрать произвольную дату относительно которой будет высчитываться текущий возраст (в ранних версиях текущий возраст считался относительно даты запуска программы)

Версия 1.09. Исправлено некорректное отображение дат при генерации документов из шаблона. В некоторых случаях дата отображалась с добавлением часов-минут-секунд.

Версия 1.08. Добавлена возможность сравнения и объединения 2 таблиц по выбранным колонкам. Нахождение общих значений, нахождение уникальных значений которых нет в другой колонке и т.п.
