Новости

Доступна для скачивания Веста 1.56 – Добавлена новая функция Обработка отчетов с помощью которой можно суммировать данные из большого количества отчетов в один путем сложения значений ячеек.При этом создается файл в котором можно проверить как получилось то или иное число в итоговом файле с указанием названия исходного файла и строки откуда были взяты данные. При работе функции Слияние файлов название файла откуда были взяты данные теперь отображается в первой колонке, а не в последней.

Доступна для скачивания Деметра 1.8 – Добавлено создание срезов по возрасту, году рождения, полу, образовательной программе для стандартного и настраиваемого отчета. Улучшена структура папок по которым раскладываются файлы.

Доступна для скачивания Веста 1.55 – Исправлена ошибка при разделении таблицы когда в колонке есть 2 одинаковых значения отличающихся только регистром. Улучшена обработка ошибок возникающих при попытке обработать поврежденные файлы xlsx.

Доступна для скачивания Деметра 1.7 – Добавлен новый тип обрабатываемых колонок Список_ для множественных значений. Добавлено создание сводных таблиц в разрезе возраста, группы, образовательной программы. Файлы связанные с ЕГИССО и дополнительные сводные таблицы сохраняются по папкам.

Доступна для скачивания Деметра 1.6. Реализовано создание полноценных файлов для ЕГИССО, с автоматическим заполнением колонок: LMSZID,categoryID,ONMSZCode и т.д. Реализовано создание отдельных списков получателей льгот по группам.

На сайт выложена программа Деметра 1.4 предназначенная для автоматизации создания отчетности для ПОО (профессиональная образовательная организация).
