Создание социоматриц

Создается результирующая социоматрица по всем вопросам использованным в социометрическом исследовании.

Создается социоматрица на каждый вопрос по отдельности.

Обработка производится для вопросов как с положительным, так и негативным выбором.

Подсчитывается количество полученных, сделанных и взаимных выборов.

Подсчитываются групповые индексы: Индекс групповой сплоченности,Индекс групповой конфликтности (при наличии вопросов на негативный выбор),Индекс референтности группы .

Подсчитываются индивидуальные индексы: Индекс социометрического статуса, Индекс эмоциональной экспансивности, Коэффициент удовлетворенности.

Создается проверочный файл в котором указаны все выборы участников исследования для каждого вопроса с подсчетом количества выборов.

Создается файл в котором будут указаны дубликаты ФИО(при наличии) это полезно при проведении социометрии с помощью Яндекс Форм или аналогов.

В случае если выбирались люди отсутствовавшие при проведении социометрии, для каждого вопроса будут создаваться файлы в которых указывается кто и сколько раз выбрал отсутствовавших.

Создание социоматриц.

По каждому вопросу социометрического исследования создается 7 вариантов социограммы

Круги обозначающие людей делятся по цветам и размерам в соответствии с количеством полученных выборов

Используется 5 категорий: 1) Лидеры (получившие максимум выборов), 2) Предпочитаемые (количество выборов выше среднего), 3) Пренебрегаемые (количество выборов ниже среднего), 4) Оттесненные (1-2 выбора), 5) Изолированные (0 выборов)

Взаимные выборы обозначаются стрелками зеленого цвета, односторонние выборы обозначаются стрелками синего цвета

Базовый групповой и индивидуальный анализ социометрии.

Подсчитываются количество участников, общее количество выборов, общее количество взаимных выборов с вычислением процентов, среднее количество выборов, плотность графа (доля выборов от максимально возможных в группе).

Создается список диад (взаимных выборов).

Создается список звездообразных структур (людей имеющих 3 и более связей) с указанием кто выбрал и кого выбрали.

Создается список изолированных узлов (участников без взаимных выборов) с указанием кто выбрал и кого выбрали.
